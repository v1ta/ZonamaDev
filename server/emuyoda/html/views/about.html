About EmuYoda

EmuYoda is a Web Service and API layer to the ZonamaDev SWGEmu Server Development System

The underlying technology consists of an AngularJS client that connects to the server-side YodaAPI. The YodaAPI is implemented in Lua using the OpenResty distribution of Nginx.

The goal of EmuYoda is to empower users to manage their servers while also encouraging web developers to learn more about Lua and the SwgEmu server system.
